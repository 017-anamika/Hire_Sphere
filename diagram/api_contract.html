Hire Sphere - API Contract

Authentication API /api/auth
POST /signup

Registers a new user.

Request Body:

{ "name": "string", "email": "string", "password": "string" }

Success Response (201 Created):

{ "token": "string (JWT)", "user": { "id": "string", "name": "string" } }
POST /login

Authenticates a user and returns a token.

Request Body:

{ "email": "string", "password": "string" }

Success Response (200 OK):

{ "token": "string (JWT)", "user": { "id": "string", "name": "string" } }

Error Response (401 Unauthorized):

{ "error": "Invalid credentials." }
User Profile API /api/users
GET /:id

Retrieves the public profile for a specific user.

Success Response (200 OK):

{ "id": "string", "name": "string", "headline": "string", "summary": "string", "experiences": [], "education": [] }
PATCH /me (Auth Required)

Updates the profile of the currently authenticated user.

Request Body:

{ "headline": "string (optional)", "summary": "string (optional)" }

Success Response (200 OK):

{ /* Full updated user profile object */ }
